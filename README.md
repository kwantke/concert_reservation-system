# 콘서트 예매 시스템 (Concert Reservation System)

## 프로젝트 개요
콘서트 예매 시스템은 사용자가 콘서트 좌석을 예매하고 결제 하는 REST API 기반 서비스 입니다.
<br>
Kubernetes + Argocd + Jenkins Pipline 으로 배포 자동화를 구축하였고, 
<br>
Prometheus 와 Grafana를 통해 Kubernetes 운영 관리을 수행할 수 있습니다.

## 기술 스택
- Spring Boot 3.2.9
- Spring Data JPA
- Java 17
- Gradle
- MySQL 8.0
- Docker
- Redis
- Kafka
- K6
- Junit5
- swagger
- Jenkins Pipline
- Kubernetes
- Argocd
- Prometheus
- grafaka
- IntelliJ Http Request
